# UAlly Frontend

React frontend for the UAlly task management app.

## Prerequisites

- Node.js (v16 or higher)
- npm or yarn

The frontend expects the backend running at `http://localhost:8080`.
You can change this in `src/config.ts`.

## Development

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm start
```

The app runs at `http://localhost:3000`.

## Build

```bash
npm run build
```

Build output is in `build/`.
